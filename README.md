# Medical-Chatbot-with-LLMs-LangChain-Pinecone-Flask-AWS

# How to run?
### STEPS:
Clone the repossitory
```bash
git clonehttps://github.com/Sayantika2327/Medical-Chatbot-with-LLMs-LangChain-Pinecone-Flask-AWS.git
```
###  STEP 01- Create a conda environmet after opening the repository

```bash
conda create -n medical-chatbot python=3.10 -y
```

```bash
conda activate medibot
```

### STEP 01- install the requirements
```bash
pip install -r requirements.txt
```
